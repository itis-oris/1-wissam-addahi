Library Management System Overview

About the System

A software system designed for library employees. It ensures the storage of information about the books available in the library, its readers, and its reading rooms.

Books

The database should store the following details for each book:

  • Book title
  • Author(s)
  • Publisher
  • Year of publication
  • Number of copies of the book in each library room
  • Book code
  • Date the book was assigned to a reader

Readers

Information about library readers should include:

  • Library card number
  • Readerâ€™s last name
  • Passport number
  • Date of birth
  • Address
  • Phone number
  • Education level
  • Academic degree

Readers are assigned to specific rooms and can register and unregister from the library.

Reading Rooms

Each room is characterized by:

  • Room number
  • Name
  • Capacity (number of people who can simultaneously work in the room)

Library Operations

The library can:

  • Acquire new books and discard old ones.
  • Change a book's code due to reclassification.
  • Change a library card number due to re-registration.

Required Librarian Information

Librarians may need the following information:

  • Which books are assigned to a specific reader?
  • What is the title of the book with a given code?
  • What is the code of the book with a given title?
  • When was a book assigned to a reader?
  • Which readers borrowed books more than a month ago?
  • Which readers have been assigned books where the number of copies in the library does not exceed two?
  • How many readers are currently using the library?
  • How many library readers are under 20 years old?
  • What percentage of readers have primary, secondary, higher education, or an academic degree?

Librarian Operations

Librarians should be able to:

  • Register a new reader in the library.
  • Remove readers from the list who registered in the library more than a year ago and have not re-registered.
  • Remove old or lost books.
  • Add a book to the library's collection.

Reports

The system should provide the following reports:

  • A report on the number of books by a specific author in a reading room.
  • A monthly report on library activity, including:
    • The number of books and readers in each room and the library as a whole on the current day.
    • The number of readers registered in the library during the reporting month.
    • Which books were borrowed during the month and how many times.
    • Readers who did not borrow any books.
